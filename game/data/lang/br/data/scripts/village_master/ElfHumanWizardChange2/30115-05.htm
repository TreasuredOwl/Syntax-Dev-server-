Cada mark requer seu próprio teste.
A Mark of the Scholar é reservada para aqueles que passam na Trial of the Scholar. Você deve estar acima do nível 35 para se qualificar. Para fazer este teste, visite Magister Mirien no templo da Town of Dion.
A Mark of the Trust é reservada para aqueles que passam no Testimony of the Trust. Você deve estar acima do nível 37 para se qualificar. Para fazer este teste, visite High Priest Hollint na Town of Oren.
E por fim, a Mark of the Magus é reservada para aqueles que passam no Testof the Magus. Você deve estar acima do nível 39 para se qualificar. Para fazer este teste, visite Bard Rukal da Town of Dion.
