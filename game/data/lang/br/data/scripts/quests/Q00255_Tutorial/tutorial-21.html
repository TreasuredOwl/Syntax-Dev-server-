[Transferência de Classe de Human Fighter]:
Um Fighter que atingiu o nível 20 pode realizar uma transferência de classe para Warrior, Knight ou Rogue. Um personagem pode iniciar a missão de transferência de classe no nível 18; se for bem-sucedido, poderá usar habilidades mais especializadas e poderosas do que antes. Para se tornar um Warrior, você deve primeiro encontrar Master Auron. Para se tornar um Knight, você deve primeiro encontrar Sir Klaus Vasper. E para se tornar um Rogue, você deve primeiro encontrar Captain Bezique. Todos podem ser encontrados em Gludin Village. Pense cuidadosamente sobre qual classe deseja escolher e então procure o NPC apropriado.
