Usando a Roda do Mouse

Bom trabalho. Agora você aprenderá sobre ampliar e reduzir a tela usando a roda do mouse.
Se você girar a roda do mouse para baixo, você amplia a tela do jogo. Gire a roda para cima para reduzir a tela.
Quanto mais você girar a roda, maior ou menor a tela ficará, mas note que há um limite para o quanto você pode ampliar ou reduzir.
