Newbie Helper:
Bem-vindo! Você está pronto para uma missão?
Você viu os Gremlins por aqui? Eles roubaram a preciosa Blue Gemstone!
Você deve recuperá-la deles!
Coloque o cursor sobre um Gremlin e clique com o botão esquerdo. O cursor mudará para uma espada. Clique novamente com o botão esquerdo para atacar.
Antes de atacar, para aumentar a força do seu ataque, use a magia Soul Cry pressionando a tecla F2. Pressione novamente para desativar a magia.

Você será bem recompensado por seus esforços. Boa sorte.
