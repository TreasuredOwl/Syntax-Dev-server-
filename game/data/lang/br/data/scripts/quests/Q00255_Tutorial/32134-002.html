Newbie Helper:
Você esqueceu sua missão? Você deve derrotar os Gremlins e recuperar a Blue Gem que eles roubaram.
Para selecionar um Gremlin para atacar, simplesmente mova o cursor sobre ele e clique com o botão esquerdo do mouse. O cursor mudará para o formato de uma faca; quando isso acontecer, clique novamente com o botão esquerdo do mouse para começar a atacar.
Se você conseguir me trazer a Blue Gem de volta, eu lhe darei uma recompensa!
