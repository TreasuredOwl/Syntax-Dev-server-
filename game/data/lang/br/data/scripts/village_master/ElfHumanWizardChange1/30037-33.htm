High Priestess Levian:
Você precisa do nível apropriado e do token para se tornar um Elven Oracle. Para mudar de profissão, seu nível deve ser pelo menos 20, e para se tornar um Oracle, você deve ter o token que comprova que passou no Test of the Elven Oracle.
Você atingiu um nível suficiente, mas para se tornar um Oracle, deve passar no teste e trazer a leaf of the Oracle como prova. Para fazer o Test of the Oracle, vá até Priest Manuel, que pode ser encontrado no santuário da cidade de Gludio. Para chegar à cidade de Gludio, siga pela estrada para o leste. Você deve passar no teste dele e trazer de volta a leaf of the Oracle.
